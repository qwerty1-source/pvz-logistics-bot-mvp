\# 📦 PVZ Logistics Bot (MVP)

Telegram-бот для автоматизации учёта логистических событий на пунктах выдачи заказов (ПВЗ).



\## 🎯 Цель

Убрать ручной контроль чатов, структурировать данные по транспорту и приёмке, автоматически выгружать отчёты в Google Sheets и отправлять уведомления контролёру.
